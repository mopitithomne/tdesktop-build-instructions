# Telegram Desktop Build Instructions

1) Run [prepare.sh](prepare.sh) to download the official tdesktop repository

    *If it exists already, It will update the source code*

2) Run [build.sh](build.sh) to build the docker image

3) Run [run.sh](/run.sh) to start the compilation of Telegram Desktop

    **Pass your APP_ID and API_HASH as argument of the run command**
